Aplicativo Python/Django com interface para:
- cadastramento de ilustradores
- cadastramento de projetos
- cadastramento de componentes curriculares
- cadastramento de créditos
- cadastramento de ilustrações
- gerenciamento da produção de ilustrações

PASSO 1: Configurar os grupos de usuários em /admin
    - Administrador
    - Coordenador
    - Comum
PASSO 2: adicionar usuário admin ao grupo Administrador
PASSO 3: criar um usuário coordenador e adicioná-lo ao grupo Coordenador. Ele será responsável por criar os demais usuários.
